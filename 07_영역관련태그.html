block 형식 태그

    - 화면(공간, 영역)을 수직 분할
      (세로 화면을 가로로 잘라서 층을 나눔 , 레고 블럭)

    - 한 층(행)을 모두 차지함

    - width/height 크기 지정 속성 적용 가능

    - block 형식 요소 종류
     h1~h6(제목),  p/pre(문단), hr(수평선),
     div(영역 분할), 시맨틱 태그(div태그에 이름을 추가한 태그)

h3는 block 형식 요소 입니다

    block 형식은 작성된 내용이 짧더라도
    한 줄을 모두 차지하려는 성질 때문에
    다음 태그 내용을 다음 줄에 출력됩니다

두 번째
세 번째

in line 형식 요소

    - 화면을 수평 분할
      (가로 화면/영역을 세로로 잘라서 열로 분할, 김밥 썰기)

    - 작성된 내용 만큼의 크기만 차지함

    - width/height 크기 지정 속성 적용 x

    - inline 형식 요소 종류
      b(굵게), i(기울임), strong/em(강조)
      mark(형광펜), span(영역 분할)
inline 형식은 행을 세로로 분할하는 성질을 가지고 있어 줄이 바뀌지 않고 계속 옆으로 작성된다
첫 번째 두 번째 세 번째

block vs inline 영역 지정 방식 차이

  • block 형식 : 사각형 형태로 영역을 지정
  • inline 형식 : 작성된 내용(content)단위로 영역을 지정

inline-block 형식 요소

    - inline 특징 중 수평 분할(옆으로 나열됨)
    - block 특징 중 크기 저장 가능

    위 둘을 섞은 형식 == inline-block

    - 대표적인 태그 : img , iframe
